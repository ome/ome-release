  • Please note that this version of FLIMfit is compatible with OMERO 5.1 servers. If you require compatibility with an OMERO 5.0 server, earlier versions are available from the main downloads folder.

  • Mac users, please note that in order to run FLIMfit you will need the correct MCR (MATLAB Compiler Runtime) installed on your system (currently 2014b, v8.4) which enables the execution of compiled MATLAB applications or components on computers that do not have MATLAB installed. If you do not, or are in any doubt, please also download the MCR for your platform below. The Windows Installer will automatically install the MCR. The link below to the Windows MCR is provided only in case of problems.
